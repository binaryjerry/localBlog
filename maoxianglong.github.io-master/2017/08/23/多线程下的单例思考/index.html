一、导论

逛论坛看到一个耐人寻味的问题，单例模式在多线程下会产生什么有趣的问题，花了半个小时玩了玩，在此记录一下。

二、单线程下的单例模式

单线程下的单例模式不用细说，随便跑，总是同一个实例，不会产生两个不同的实例，不想多解释了。
